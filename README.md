# Dynamics Lab

MODT Research Dashboard - A Next.js application showcasing AMD Ryzen mini PC motherboard research data.

## Features

- **Data Explorer**: Filter and browse TOPC/Topton boards by type (Gaming, NAS, Firewall)
- **Configurator**: Interactive workload selector with budget slider
- **Architecture Blueprints**: Visual system architecture comparisons
- **Prof's Verdict**: Top recommendation with technical specifications

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
